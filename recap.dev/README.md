## Run recap.dev-server

```bash
$ docker-compose up -d
```

## Run app

```bash
# change directory
$ cd app

# install libs
$ npm install

# run development mode
$ npm run start

# run watch mode
$ npm run start:dev

# run production mode
$ npm run start:prod
```

## How-to

1. Recap.Dev UI starts on http://localhost:8081
2. App API documentation starts on http://localhost:3000/docs
